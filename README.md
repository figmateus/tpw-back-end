## Projeto Back-end 

<p>Projeto back-end da disciplina de tpw utilizando nodejs + express e MySql</p>

## Instalando o projeto 

* Clone o projeto
```
git clone https://github.com/figmateus/tpw-back-end.git
```

* Instale as dependencias node
```
npm install
```

* rode o projeto

```
node .
```
